# Web Design Helper

Webページの画面設計書作成を支援するChrome拡張機能です。ページ上の要素を選択して枠と番号を付け、PNG画像として出力できます。

## 機能

- **要素の選択**: ページ上の要素をクリックして選択、枠と番号バッジを表示
- **ラベル編集**: 連番の自動付与、任意のラベル（「1-1」「A」など）に変更可能
- **枠色の変更**: 5色（赤・青・緑・黄・黒）から選択
- **PNG出力**: ページ全体のスクリーンショットをダウンロード
- **クリップボードコピー**: 画像をクリップボードに直接コピー
- **状態の保存**: URLごとに編集状態を自動保存・復元

## インストール

```bash
npm install
npm run build
```

1. Chrome で `chrome://extensions` を開く
2. 「デベロッパーモード」を有効化
3. 「パッケージ化されていない拡張機能を読み込む」から `dist` フォルダを選択

## 使い方

1. **編集モードの開始**
   - ページ上で右クリック →「画面設計ヘルパーを開始」を選択
   - または拡張機能アイコンをクリック

2. **要素の選択**
   - ページ上の要素をクリックして選択
   - 選択した要素に枠と番号バッジが表示される

3. **画像の出力**
   - 「PNG出力」でファイルをダウンロード
   - 「コピー」でクリップボードにコピー

4. **編集モードの終了**
   - パネル右上の「×」ボタンをクリック

## 開発

```bash
npm run dev    # 開発モード（ファイル監視）
npm run build  # プロダクションビルド
```
